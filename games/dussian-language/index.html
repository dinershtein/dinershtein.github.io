📜 Причастие

Определение

Причастие — особая форма глагола, которая обозначает признак предмета по действию и отвечает на вопросы какой? какая? какое? какие?
Мальчик, читающий книгу, сидел у окна.
читающий ← читать (глагол) + какой? (вопрос прилагательного)

Признаки глагола

ПризнакОписаниеПример
ВидСовершенный / несовершенныйчитающий — прочитавший
ВремяНастоящее / прошедшеечитающий — читавший
Возвратность-ся/-сьулыбающийся
ЗалогДействительный / страдательныйчитающий — читаемый

Признаки прилагательного

ПризнакПример
Родчитающий, читающая, читающее
Числочитающий — читающие
Падежчитающий, читающего, читающему...
Полная/краткая формапрочитанный — прочитан

Виды причастий и суффиксы

Действительные (сам делает)

ВремяСуффиксыПримеры
Настоящее (I спр.)-ущ-/-ющ-читающий, пишущий
Настоящее (II спр.)-ащ-/-ящ-дышащий, видящий
Прошедшее-вш-/-ш-читавший, нёсший

Страдательные (над ним делают)

ВремяСуффиксыПримеры
Настоящее (I спр.)-ем-/-ом-читаемый, ведомый
Настоящее (II спр.)-им-видимый, слышимый
Прошедшее-нн-/-енн-/-т-прочитанный, решённый, взятый

🎭 Деепричастие

Определение

Деепричастие — особая форма глагола, которая обозначает добавочное действие и отвечает на вопросы что делая? что сделав? как?
Улыбаясь, она вошла в комнату.
вошла (как? что делая?) улыбаясь — два действия одновременно

Признаки глагола

ПризнакПример
Вид (сов./несов.)читая — прочитав
Возвратностьулыбаясь, умывшись
Переходностьчитая (что?) книгу

Признаки наречия

Неизменяемость — не изменяется по родам, числам, падежам.

Синтаксическая роль — всегда обстоятельство.

Образование

ВидВопросСуффиксыПримеры
Несовершенныйчто делая?-а/-ячитая, крича
Совершенныйчто сделав?-в/-вши/-шипрочитав, умывшись

Важное правило!

⚠️ Деепричастие и глагол-сказуемое должны относиться к одному подлежащему.
❌ Подъезжая к станции, у меня слетела шляпа.
Ошибка! Шляпа не может подъезжать.
✓ Подъезжая к станции, я потерял шляпу.
Верно! «Я» и подъезжал, и потерял.

⚖️ Сравнение

Причастие vs Деепричастие

📜 Причастие

  • Вопрос: какой?
  • Обозначает признак
  • Изменяется
  • Роль: определение
  • Относится к существительному

🎭 Деепричастие

  • Вопрос: что делая? как?
  • Обозначает добавочное действие
  • Не изменяется
  • Роль: обстоятельство
  • Относится к глаголу

Примеры

Читающий мальчик сидел у окна.
Причастие: мальчик (какой?) читающий
Мальчик сидел, читая книгу.
Деепричастие: сидел (как?) читая

✍️ Правописание

Н и НН в причастиях

Пишется НН:

УсловиеПример
Есть приставка (кроме не-)прочитанный, закрытый
Есть зависимое словокрашенный маслом пол
Совершенный видрешённый (решить — сов. вид)
На -ованный/-ёванныймаринованный, корчёванный

Пишется Н:

УсловиеПример
Краткие причастиякнига прочитана, задача решена
Нет приставки, зав. слова, несов. видкрашеный пол, вязаный шарф
⚠️ ИСКЛЮЧЕНИЯ с НН (всегда пишутся с НН):

желанный священный нечаянный невиданный неслыханный нежданный негаданный деланный медленный чванный чеканный жеманный данный виданный слыханный считанный
⚠️ ИСКЛЮЧЕНИЯ с Н (всегда пишутся с Н):

смышлёный названый (брат) посажёный (отец) приданое кованый жёваный клёваный

Суффиксы причастий по спряжению

СпряжениеДействительные наст. вр.Страдательные наст. вр.
I спряжение-ущ-/-ющ--ем-/-ом-
II спряжение-ащ-/-ящ--им-
⚠️ ИСКЛЮЧЕНИЯ — глаголы I спряжения (суффиксы -ущ-/-ющ-, -ем-):

брить → бреющий стелить → стелющий зиждиться → зиждущийся зыбиться → зыблющийся
⚠️ ИСКЛЮЧЕНИЯ — глаголы II спряжения (суффиксы -ащ-/-ящ-, -им-):

гнать → гонящий держать → держащий дышать → дышащий слышать → слышащий
смотреть → смотрящий видеть → видящий ненавидеть → ненавидящий
обидеть → обидящий терпеть → терпящий зависеть → зависящий вертеть → вертящий

НЕ с причастиями

Слитно:

УсловиеПример
Без НЕ не употребляетсянегодующий, ненавидящий, недоумевающий
Нет зависимых слов и противопоставлениянепрочитанная книга
Есть наречия меры и степенисовершенно непродуманное решение

Раздельно:

УсловиеПример
Есть зависимое словоне прочитанная мной книга
Есть противопоставление с «а»не прочитанная, а просмотренная
Краткие причастиякнига не прочитана
С усилительными отрицаниямидалеко не, вовсе не, отнюдь не

НЕ с деепричастиями

Правило: НЕ с деепричастиями пишется раздельно: не читая, не прочитав, не думая.
⚠️ ИСКЛЮЧЕНИЯ (слитно) — без НЕ не употребляются:

негодуя недоумевая ненавидя невзлюбив недомогая неистовствуя нежничая неволя

Суффиксы деепричастий

ВидСуффиксыПримеры
Несовершенный-а/-ячитая, крича, улыбаясь
Совершенный-в, -вши, -шипрочитав, умывшись, принёсши
⚠️ ИСКЛЮЧЕНИЯ — глаголы без деепричастий несов. вида:

Не образуют деепричастия несов. вида глаголы:
• с основой на шипящий + -ну-: пахнуть, мёрзнуть, тянуть
• без гласных в основе наст. вр.: бить (бьют), пить, лить, шить, жечь, ткать
• односложные: ждать, лгать, рвать, звать, врать
• на -чь: беречь, мочь, стеречь, печь

Гласные перед -Н-/-НН- в причастиях

Инфинитив наПеред -Н-/-НН-Пример
-ать, -ятьА / Янаписать → написанный, посеять → посеянный
-ить, -етьЕпостроить → построенный, увидеть → увиденный
⚠️ ИСКЛЮЧЕНИЯ:

выровненный (от выровнять) выровнянный (от выровнить)
замешенный (тесто) замешанный (в преступлении)
развешенный (по весу) развешанный (на стене)
